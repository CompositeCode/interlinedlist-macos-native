InterlinedList for Mac Alpha

Version 0.0.1 — Released July 3, 2026 — Requires macOS 15.0 or later

First alpha release of InterlinedList for Mac. This build is intended for early testers and may contain bugs. Please report issues to the InterlinedList team.

What’s included

  • Messages — timeline, compose, reply, repost
  • Lists — browser, list detail, rows, schema editor, watchers, connections graph
  • Documents — editor with live preview and sync
  • Social — profile, roster, follow/unfollow, pending requests
  • Notifications — inbox with approve/reject actions
  • Settings — account, linked accounts, avatar upload
  • Exports — CSV export for messages, lists, list data rows, and follows
  • Organizations — member management and role assignment
  • Scheduled posts
  • Automatic updates via Sparkle

Known limitations

  • Alpha build — expect rough edges
  • Some upstream API features are pending server-side availability
