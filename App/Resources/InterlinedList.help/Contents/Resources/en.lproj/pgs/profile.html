Viewing a profile

Select Profile in the sidebar and enter a username. The profile header shows the user's avatar, display name, and handle.

What's shown today

In this release the profile header is intentionally minimal. The following fields are visible:

  • Avatar.
  • Display name.
  • Username (handle).

These are coming in a future update:

  • Bio.
  • Date joined.
  • Private-account indicator.
  • Follower and following counts.
Why some profiles appear empty. A profile is shown using information attached to the user's most recent public message. If a user has never posted a public message, the app cannot show their profile right now and you see a friendly "This profile has no public messages yet" message. This limit will lift when richer profile data becomes available; until then, the empty state is the expected behavior, not an error.

Coming in a future update

  • Following and unfollowing users.
  • Viewing follower and following lists.
  • Approving or rejecting follow requests for private accounts.
  • Mutual-follow lists.
