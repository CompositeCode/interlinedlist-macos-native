Browsing public lists

Back to Help home

Anyone can view a user's public lists. You do not need to follow the user first.

Look up a user's lists

  1. Select Lists in the sidebar.
  2. Type the username you want to browse into the username field.
  3. The user's public lists appear in the content pane.

Open a list

Click a list to see its rows in the detail pane. Rows are shown with the data the list owner chose to make public. Row layout follows the list's schema; columns may differ from list to list.
