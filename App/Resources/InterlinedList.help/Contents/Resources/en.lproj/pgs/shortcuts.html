Keyboard shortcuts

Back to Help home

Application

  • ⌘, — open the Settings window.
  • ⌘Q — quit InterlinedList.
  • ⌘? — open this help book.

Composing

  • ⌘N — open the composer window for a new post. Available when M2 ships.
  • ⌘⏎ — publish the message from inside the composer. Available when M2 ships.

Window

  • ⌘W — close the front window.
  • ⌘M — minimize the front window.

Coming in a future update

  • Sidebar shortcuts ⌘1 through ⌘7 for Timeline, Scheduled, Notifications, Lists, Documents, Organizations, and Profile.
  • File menu shortcuts for New List, New Document, and Export.
