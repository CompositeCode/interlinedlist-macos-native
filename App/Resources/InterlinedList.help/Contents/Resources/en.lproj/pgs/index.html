InterlinedList Help

Welcome to InterlinedList for macOS. This help book explains what the app can do today and how to use it. Features still in development are marked coming in a future update.

Topics

  • Getting started — sign in and a tour of the main window.
  • Reading the timeline — feeds, tags, and message threads.
  • Composing and reacting — write posts, reply, repost, dig. Available when M2 ships.
  • Browsing public lists — view any user's public lists.
  • Profiles — viewing a user's profile and its current limits.
  • Keyboard shortcuts — every command bound to a key today.
  • Troubleshooting — sign-in, empty timelines, and known limits.

What's in this release

The current build ships the read-only core: timeline, message threads, public list browsing, and a thin user profile. Composing, replies, reactions, reposts, full list editing, documents, notifications, organizations, and exports are coming in a future update. See Known limits for the current feature boundary.
