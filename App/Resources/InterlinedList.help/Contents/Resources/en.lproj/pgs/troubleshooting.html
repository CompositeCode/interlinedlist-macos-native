Troubleshooting

Back to Help home

I can't sign in

  • Double-check that the same email and password work at interlinedlist.com. The app uses the same account.
  • If the website shows your account is locked or unverified, finish those steps on the site first.
  • Use Reset password on the welcome window if you have forgotten your password.
  • If your network requires a proxy or VPN, make sure macOS can reach interlinedlist.com.

A profile says "no public messages yet"

This is expected. In the current release the app builds a profile from the user's most recent public message. Users who have never posted a public message cannot be shown as a profile yet. The user does exist; their profile will become viewable as soon as they post a public message, or when a future update adds direct profile lookup.

The timeline is empty

  • If you have just signed in, scroll up to trigger a refresh.
  • If the Mine scope is empty, you have not posted any messages yet on this account. (Composing arrives in the next milestone.)
  • If a tag filter is set, clear it.

A message thread won't open

Threads require an internet connection to load replies for the first time. Check your network and try again. If the original message has been deleted by its author, the thread cannot be opened.

Known limits in this release

The following features are coming in a future update and are not available yet:

  • Composing posts, replies, reposts, "I Dig!" reactions, editing, and deleting your own messages.
  • Creating, editing, and sharing your own lists; list schemas; nested lists; the connections graph; GitHub-backed list refresh.
  • Documents, the document editor, and offline sync.
  • Following and unfollowing users; follower and following lists; mutual follows; private-account requests.
  • Notifications (sidebar badge, tray, and system notifications).
  • Organizations and member management.
  • Media attachments and scheduled posts.
  • OAuth identity linking (GitHub, Mastodon, Bluesky, LinkedIn).
  • CSV exports.
  • Settings polish: email change, account deletion, avatar upload.

Reporting an issue

If something the help book says should work doesn't, please file an issue on the project's repository with the steps you took, what you expected, and what you saw. Do not share your account credentials.
