# Getting started

**Audience:** application end users.

This page is the source of truth for the Help Book's "Getting started" page (`App/Resources/InterlinedList.help/Contents/Resources/en.lproj/pgs/getting-started.html`). Keep the wording in sync.

## Before you sign in

InterlinedList for macOS connects to your existing account at [interlinedlist.com](https://interlinedlist.com). If you don't have one yet, you can create it from the welcome window with **Create account**, or sign up on the website first.

## Sign in

1. Launch InterlinedList. The welcome window asks for your email and password.
2. Enter your credentials and click **Sign in**.
3. The app stores your access token in the macOS Keychain. You stay signed in between launches.

> **Note.** If you forget your password, click **Reset password** on the welcome window and follow the email link.

## The main window

The main window uses the standard macOS three-pane layout:

- **Sidebar** — switch between Timeline, Lists, Profile, and other sections.
- **Content list** — the items in the selected section.
- **Detail / inspector** — the selected item's full content.

## What you can do right now

- Read your timeline.
- Open a message thread to see its replies.
- Browse any user's public lists by username.
- View a basic profile header for any user who has at least one public message.
